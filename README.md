**Project title: **University Student Performance Analytics System.

**Description:**The University Student Performance Analytics System is a data-driven decision-support platform designed to analyze, monitor, and predict academic performance of university students using structured educational data. The system integrates student academic records, attendance data, assessment scores, and behavioral indicators to generate meaningful insights that support students, faculty members, and academic administrators.
